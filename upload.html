How to Upload Resources

1. Access the GitHub Repository

Visit the repository: Atomi1cDoc/Atomi1cDoc.github.io

2. Upload Files

  1. Navigate to the appropriate folder (resources/ibdp or resources/igcse)
  2. Click "Add file" â†’ "Upload files"
  3. Drag and drop your files or click "choose your files"
  4. Commit changes directly to the main branch

3. Update resources.json

Edit resources.json to include metadata for new files:

{
  "ibdp": [
    {
      "name": "New_File.pdf",
      "size": "1.5MB",
      "date": "2023-10-15"
    }
  ]
}

File Requirements

  • Max file size: 20MB
  • Allowed formats: PDF, DOCX, PPTX, ZIP
  • File names should use underscores instead of spaces
